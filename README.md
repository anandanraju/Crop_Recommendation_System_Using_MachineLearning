# Crop_Recommendation_System_Using_MachineLearning
Crop Recommendation system using Machine Learning that predicts crop suitability by factoring all relevant data such as temperature, rainfall, location, and soil condition. 

This system is primarily concerned with performing Agro Consultant's principal role, which is to provide crop recommendations to farmers.
